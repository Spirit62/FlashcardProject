AS-Level Flashcard Master
Live Demo: https://spirit62.github.io/portfolio

A high-performance, structured study tool designed for students tackling the CAIE AS-Level curriculum. This application focuses on rapid recall and logical understanding of core STEM concepts.

🚀 Features
Subject-Specific Modules - Dedicated interfaces for Mathematics, Physics, and Chemistry.

Semantic Architecture - Optimized with HTML5 <article> and <main> tags for accessibility.

Dynamic CSS Variables - Theme-based styling using :root for consistent UI across subjects.

Fluid Navigation - Intuitive "Back" navigation and standardized card controls.

Responsive Design - Built with Flexbox to ensure a seamless study experience on mobile or desktop.

Focus-Oriented UI - Clean, distraction-free interface to maximize retention.

🎨 Design
Professional Palette - Utilizes a "Deep Sea" and "Soft Blue" scheme for reduced eye strain.

Visual Consistency - Standardized button widths and high-contrast typography.

Interactive States - Smooth CSS transitions on hover and active states.

Typography - Leverages the Montserrat font family for a modern, executive feel.

🛠 Tech Stack
Frontend: Pure HTML5, CSS3, JavaScript (Vanilla).

Styling: Modern CSS with Flexbox and custom properties (variables).

Hosting: GitHub Pages.

📚 Modules (In Progress)
Mathematics - Focus on Quadratic formulas, Circular Measure, and Trigonometry.

Physics - Mechanics, Waves, and Electricity (Coming Soon).

Chemistry - Organic Chemistry and Stoichiometry (Coming Soon).

Computer Science - Logic Gates and Pseudocode (Coming Soon).

🚀 How to Use
Dashboard: Select your target subject from the executive portfolio main page.

Study Mode: View the question on the primary card.

Navigation: Use the Next and Previous buttons to cycle through the question bank.

Return: Use the Back button to switch subjects at any time.

📈 Future Roadmap
[ ] 3D Card Flip: Implementing CSS transforms for a realistic card interaction.

[ ] JSON Integration: Transitioning from hardcoded text to dynamic JSON data fetching.

[ ] Progress Tracking: LocalStorage implementation to save session data.

[ ] Dark Mode: High-contrast theme for late-night study sessions.
